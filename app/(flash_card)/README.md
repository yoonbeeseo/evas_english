# 단어암기 플래쉬 카드

1. 단어장 리스트

   - 내 단어장에 담기
   - 내 단어장에서 삭제
   - 등록된 반에 단어장이 설정되어 있다면 제일 첫 번째 아이템으로 보이게 하기
   - 나머지 단어장도 원하면 볼 수 있음

2. 단어장 카드 보기

   - 단어
   - 뜻
   - 유의어
   - 다시 한번 암기 버튼
   - 암기 완료

3. 단어장 테스트
   - 스피스 테스트
     - 단어 보여주고 뜻 생각하기
     - 모름/다음 버튼
     - 모름 누르면 모르는 단어 리스트로 가서 다시 한번 테스트 할 수 있게 하기
     - 범위 선택할 수 있게.
     - day1 인 경우
     - day1-3 등 다중 선택인 경우
     - 몇개의 단어가 될지, 그리고 셔플해서 보여주기
     - 단어의 뜻 중에서 5-10개의 선택지를 줘 선택하기 보기 보여줘 버튼으로 아래에서 버튼 보이거나 없이 그냥 뜻 확인후 넘어가거나
   - 플래쉬 카드 형태
     - 뒤집는 애니메이션
     - 단어 뜻 유의어 있음
   - 좌 우 방향은 다음 또는 이전 단어
   - 아래의 버튼 모름은 모름 리스트 정답알고 있음 버튼은 확인 끝난 버튼
   - 결과 12/20 이런 식으로 결과 저장하기
   - 그래프 등으로 결과 확인할 수 있는 코너 만들기
